# vimfiles

## Dependencies
  - Ruby 2.0
  - Java 1.7 (if you want to use eclim)
  - Scala 2.10 (if you want to use eclim)
  - Eclipse 4.3 (if you want to use eclim)
  - ack
  - build-essential
  - cmake
  - exuberant-ctags
  - git
  - liblua5.1-0-dev
  - libluajit-5.1-dev
  - libperl-dev
  - libpython2.7-dev
  - libpython3.3-dev
  - luajit

## Installation
    $ git clone git://github.com/mpereira/vimfiles ~/.vim
    $ cd ~/.vim
    $ make install
